WESTERN CLASSICAL COMPOSERS

est une application web qui permet de visualiser et de naviguer dans l’ensemble de la base de données Open Opus.

On récupère aussi des données via l'API Wikimedia afin d'obtenir une brève description de chaque compositeur.
